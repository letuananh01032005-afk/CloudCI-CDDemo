CI/CD & Automation

Tự động hóa với GitHub Actions

Giải pháp mạnh mẽ giúp bạn xây dựng (Build), kiểm thử (Test), và triển khai (Deploy) mã nguồn tự động ngay trên kho chứa GitHub.

Tại sao nên dùng GitHub Actions?

Tích hợp trực tiếp

Không cần cài đặt công cụ bên thứ ba. Mọi quy trình CI/CD được quản lý trực tiếp ngay tại các tab công việc trong kho mã nguồn của bạn.

Kho Actions khổng lồ

Truy cập vào GitHub Marketplace với hàng ngàn Actions có sẵn do cộng đồng đóng gói để tái sử dụng, giúp tiết kiệm thời gian viết mã cấu hình.

Hỗ trợ mọi ngôn ngữ

Hoạt động mượt mà với Node.js, Python, Java, PHP, Ruby, C/C++, Android, iOS và mọi nền tảng đám mây phổ biến hiện nay.

Thành phần cốt lõi

Một chuỗi tự động hóa của GitHub Actions hoạt động dựa trên cơ chế bắt sự kiện, bao gồm các thành phần cơ bản sau:

1

Event (Sự kiện)

Là tác vụ kích hoạt chạy tự động, ví dụ như hành động: git push code mới, mở một bản Pull Request, hoặc chạy theo lịch hẹn giờ (Cron job).

2

Workflow (Quy trình)

Là một tiến trình tự động hóa tổng thể được thiết lập bằng file định dạng .yml nằm trong thư mục .github/workflows/.

3

Job (Công việc) & Step (Bước làm)

Mỗi Workflow chứa một hoặc nhiều Jobs chạy song song hoặc tuần tự. Trong mỗi Job lại chứa các Steps (các lệnh cụ thể shell script hoặc Action đóng gói sẵn) để xử lý logic.

4

Runner (Môi trường chạy)

Là các máy chủ ảo (Ubuntu Linux, Windows, hoặc macOS) do GitHub cung cấp hoặc do bạn tự cấu hình để thực thi các lệnh trong Job.

Cấu trúc một File cấu hình mẫu

.github/workflows/main.yml YAML 
# Tên của quy trình tự động hóa
name: CI Demo Workflow

# Sự kiện kích hoạt quy trình (Khi push vào nhánh main)
on:
  push:
    branches: [ "main" ]

# Danh sách các công việc cần làm
jobs:
  build-and-test:
    runs-on: ubuntu-latest # Sử dụng hệ điều hành Ubuntu máy ảo

    steps:
      # Bước 1: Lấy mã nguồn từ kho GitHub về máy ảo
      - name: Checkout source code
        uses: actions/checkout@v4

      # Bước 2: Chạy câu lệnh kiểm thử hoặc build
      - name: Run a build script
        run: |
          echo "Bắt đầu kiểm tra hệ thống..."
          echo "Quy trình CI/CD đã kích hoạt và chạy thành công!"
